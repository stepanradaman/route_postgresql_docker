# Route
Simple routing application.

## Source
* https://github.com/stepanradaman/route_postgresql

## Usege
* terminal: 
** docker pull node / postgres / kartoza/postgis / kartoza/geoserver
** cd [app folder]
** docker build -t route .
** docker-compose up -d
* wait initialization (3 min)
* browser:
** http://localhost:3000/
